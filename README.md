# Robo Speaker Mini Project

Python project that converts text to speech using Windows PowerShell and the `os` module.  
This program allows continuous input and speaks the text typed by the user.  

## Features

- Converts typed text to speech
- Continuous input until user types "stop"
- Works without any extra Python libraries
- Uses Windows built-in PowerShell speech synthesizer

## How to Run

1. Make sure you have **Python 3** installed on Windows.
2. Open terminal / VS Code terminal.
3. Navigate to the project folder.
4. Run the program:

```bash
python robospeaker_miniproject.py
